## Getting Your API Keys

### Plaid Setup
1. Go to [plaid.com](https://plaid.com) and create a developer account
2. Navigate to Dashboard > API Keys
3. Copy the following values:
   - Client ID → `PLAID_CLIENT_ID`
   - Sandbox Secret → `PLAID_SECRET` (use Development Secret for production)
4. Set `PLAID_ENV=sandbox` for development

### OpenAI Setup (for LLM)
1. Go to [platform.openai.com](https://platform.openai.com)
2. Create an account and add billing information
3. Navigate to API Keys
4. Create a new API key → `OPENAI_API_KEY`
5. Set `OPENAI_MODEL=gpt-4` or `gpt-3.5-turbo`

### JWT Secret
Generate a secure random string for `JWT_SECRET`:
```bash
# Option 1: Using Node.js
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"

# Option 2: Using OpenSSL
openssl rand -hex 64
```

## Security Notes

- **Never commit your .env file to version control**
- **Keep your API keys secure and rotate them regularly**
- **Use different keys for development and production**
- **The .env file is already in .gitignore**

## Testing Your Setup

After creating your .env file, you can test the configuration by running:
```bash
npm run dev
```

The server should start without errors if all environment variables are properly configured.
